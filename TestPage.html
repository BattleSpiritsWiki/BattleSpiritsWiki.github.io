Scáthach BotCommands

Basic Commands

ping Returns an output of pong
quote Returns a randomly selected quote from database

Moderation

ban Warning, this will ban your target if the bot has the correct permissions
kick Warning, this will kick your target if the bot has the correct permissions
unban Unbans any person who has previously been banned from this server. Requires a valid user ID

Music

play Add a song to queue
nowplaying Shows information about the current playing track
